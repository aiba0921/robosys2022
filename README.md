# robosys2022
ロボットシステム学　授業用
## plusコマンド
![test](https://github.com/aiba0921/robosys2022/actions/workflows/test.yml/badge.svg)

## 動作環境
* Ubuntu 22.04.5 LTS
* Python 3.7～3.10:テスト済み

## インストール方法
* ターミナル上で ````git clone https://github.com/aiba0921/robosys2022.git```` を実行


## コマンド概要
* 標準入力から読み込んだ数字を足す。

## 使用方法
* ````$ cd robosys2022````
* 入出力例 
  ````
   $ seq 5 | ./plus
   15
  ````
